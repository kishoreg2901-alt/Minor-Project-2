# Minor-Project-2
SpendDNA is a data analytics project built with Python, Pandas, and Google Colab that transforms raw bank transaction data into actionable financial insights. It includes expense categorization, vendor analysis, monthly trends, anomaly detection, and personalized spending reports using data analysis techniques.
SpendDNA is a Python-based data analytics project developed using Google Colab, Pandas, NumPy, and Matplotlib. The project analyzes a user's bank transaction history to generate meaningful financial insights, helping users understand their spending habits and make better financial decisions.

The system processes raw bank statements, cleans and transforms the data, categorizes expenses based on vendors, identifies spending trends, detects unusual transactions, and generates a comprehensive financial report.
Objectives
Analyze bank transaction data.
Categorize expenses into meaningful groups.
Identify top spending categories and vendors.
Track monthly spending trends.
Perform time-based spending analysis.
Detect anomalous transactions using Z-score analysis.
Generate a personalized SpendDNA report.
